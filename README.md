# 📚 Personal Library Manager (React + TypeScript)

Aplicação web para gerenciamento de biblioteca pessoal, permitindo organizar seus livros de forma simples e eficiente.

## ✨ Funcionalidades

- ✅ Adicionar livros
- ✏️ Editar livros
- 🗑️ Remover livros
- 🔍 Buscar por título ou autor
- 📊 Filtrar por status:
  - Quero ler
  - Lendo
  - Lido

## 🛠️ Tecnologias

- React
- TypeScript
- Styled Components
- React Hook Form
